PHP & SQL Based Blog Management System
Bu proje, temel içerik yönetim sistemleri (CMS) mantığıyla geliştirilmiş, dinamik bir blog platformudur. Kullanıcı kaydı, içerik oluşturma ve etkileşim özelliklerini kapsamaktadır.

Özellikler
Kullanıcı Yönetimi: register.php ve auth.php ile güvenli kayıt ve giriş (authentication) sistemi.

İçerik Yönetimi: Kullanıcıların blog gönderileri oluşturabilmesi (create_post.php) ve yorum yapabilmesi (add_comment.php).

Dinamik Veri Yapısı: PHP ile MySQL veritabanı entegrasyonu sağlanarak verilerin kalıcı ve düzenli tutulması.

Liderlik Tablosu: top_users.php ile en aktif kullanıcıların listelenmesi özelliği.

Teknik Detaylar
Backend: PHP

Veritabanı: MySQL (Şema dosyası: blog_db.sql)

Frontend: HTML & CSS (style.css)

Veritabanı Tasarımı: İlişkisel veritabanı mantığıyla kullanıcılar, postlar ve yorumlar arasındaki tablolar arası bağlantılar kurulmuştur.
